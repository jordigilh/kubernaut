# DD-GATEWAY-008: Storm Aggregation Windows

Storm detection, buffered first-alert aggregation, and sliding-window / inactivity-timeout behavior for storms were removed from Gateway. See **DD-GATEWAY-015** (storm detection logic removal) for rationale and migration notes.

Historical references to “DD-GATEWAY-008” in config or docs pointed at this topic; there is no longer a separate long-form decision file for the original storm aggregation design.
